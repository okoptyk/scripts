# Как открыть сайт у себя на компьютере

Тут всё пошагово. Просто делай по порядку, ничего сломать нельзя.

## Самый первый раз (делается один раз в жизни)

1. Нужна одна бесплатная программа, которая «оживляет» сайт. Она называется Node.
   Открой сайт https://nodejs.org и нажми большую зелёную кнопку слева
   (там будет написано LTS). Скачается файл — запусти его и просто жми «Далее»
   до конца. Это как установить любую обычную программу.

2. Открой папку с сайтом. В пустом месте внутри папки нажми правой кнопкой мыши
   и выбери «Открыть в терминале» (или «Open in Terminal»). Появится чёрное
   или синее окошко с мигающей строчкой — это нормально, так и должно быть.

3. В это окошко впиши вот такую строчку и нажми Enter:

   ```
   npm install
   ```

   Пойдут бегущие строчки — подожди минутку, пока всё остановится. Это
   компьютер скачивает то, что нужно сайту. Делается тоже только один раз.

## Каждый раз, когда хочешь открыть сайт

1. Открой папку с сайтом, нажми правой кнопкой → «Открыть в терминале».

2. Впиши строчку и нажми Enter:

   ```
   npm run dev
   ```

3. Через пару секунд появится адрес, например `http://localhost:4321/`.
   Зажми Ctrl и щёлкни по нему мышкой — откроется браузер с сайтом.
   Готово, это твой сайт!

4. Пока это окошко открыто — сайт работает. Если поменяешь текст и сохранишь,
   страница в браузере обновится сама.

## Как закрыть

В том чёрном окошке нажми одновременно Ctrl и C — сайт остановится.
Можно просто закрыть окошко.

## Если что-то выглядит сломанным

- Закрой окошко и открой заново, начни с `npm run dev`.
- Если совсем не получается — ничего страшного, просто напиши тому, кто
  помогал делать сайт. На рабочем сайте в интернете при этом ничего не
  ломается — ты у себя на компьютере, это как черновик.
